# String Functions Stage.

The *String Functions* stage is handy when we need to make string operations on a dataframe column.
The *Operation* type dropdown list contains the following Spark built-in standard string functions:

* ascii
* base64
* concat_ws
* decode
* encode
* format_number
* format_string
* title_case
* instr
* length
* lower
* locate
* lpad
* ltrim
* regexp_extract
* ubase64
* rpad
* repeat
* rtrim
* split
* substring
* substring_index
* trim
* upper
