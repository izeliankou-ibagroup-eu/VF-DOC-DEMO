# Create a Project

To create a project, you need to push the “+” button.

![](../../assets/user-guide-img/image1.png)

**Note**: this is the action of the Super-admin user only. The button is not visible for the application roles
(Viewer, Operator, Editor, Admin).

With the “+” button pushed, you get to Create Project Form to enter basic project settings:

* Project Name
* Project Description
* Requests (CPU/Memory)
* Limits (CPU/Memory)

![](../../assets/user-guide-img/image3.png)

After saving Create Project Form the project is created under the given name and then can be found on the initial
screen:

![](../../assets/user-guide-img/image4.png)
