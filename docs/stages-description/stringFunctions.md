# Stringfunctions

## Configuration Parameters

| UI Name | Key (Code) | Description |
| :--- | :--- | :--- |
| **Name** | `name` | Provide a stage name. |
| **Operation Type** | `operationType` | Select function type. |
| **Source Column** | `sourceColumn` | Column from which data is taken. |
| **Target Column** | `targetColumn` | Column where the result is displayed. |
| **Separator** | `separator` | Separator between string input values. |
| **Decimal Places** | `decimalPlaces` | A number of simbols after comma. |
| **Substring searched for** | `substringSearch` | Substring that is searched for in the sourceColumn. |
| **Position** | `position` | Position at which to start searching for a substring. |
| **Length** | `length` | Length of targetColumn. |
| **Limit** | `limit` | Limit of splitting. |
| **Delimiter** | `delimiter` | Delimiter strting for searching. |
| **Count** | `count` | Count occurrences of the delimiter. |
