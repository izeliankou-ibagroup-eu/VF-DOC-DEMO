# Filter

## Configuration Parameters

| UI Name | Key (Code) | Description |
| :--- | :--- | :--- |
| **Name** | `name` | Provide Stage name. |
| **Filter** | `filter` | Enter any boolean expression. Two or more expressions may be combined together using the logical  operators ( AND, OR ).<br><br>Examples:<br><br>> ((column1 < 10) and column2 between 10 and 25)<br><br>[More examples](https://sparkbyexamples.com/spark/spark-dataframe-where-filter/) |
