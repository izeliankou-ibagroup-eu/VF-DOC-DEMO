# Groupby

## Configuration Parameters

| UI Name | Key (Code) | Description |
| :--- | :--- | :--- |
| **Name** | `name` | Provide Stage name. |
| **Drop grouping columns** | `dropGroupingColumns` | Select checkbox to remove grouped columns from the output. |
| **Group By** | `groupBy` | Specify the key column for the operation. To specify more than one key, use a comma or Enter. |
| **Aggregate functions** | `aggregateFunctions` | Specify aggreagate functions:<br><br>1. Click on the Plus button.<br><br>2. In the drop-down lists that appear, specify the function and column name.<br><br>Available functions: Count, Avg, Min, Max, Sum, Mean.<br><br>Notes:<br><br>1. Use the Plus button to add as many columns as you need.<br><br>2. Use the Minus button to remove the filter.<br><br>3. Use the Upp Arrow and Down Arrow to change the order of columns.<br><br>[For more examples click here.](https://sparkbyexamples.com/spark/using-groupby-on-dataframe/) |
