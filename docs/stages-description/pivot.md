# Pivot

## Configuration Parameters

| UI Name | Key (Code) | Description |
| :--- | :--- | :--- |
| **Name** | `name` | Provide Stage name. |
| **Operation Type** | `operationType` | Provide operation type:<br><br>1. pivot - Transpose data from one column into multiple columns.<br><br>2. unpivot - Transpose data from multiple columns to one column. |
| **Group By** | `groupBy` | Column(s) for grouping. |
| **Aggregate function and column** | `aggregateFunc` | Aggregate function and column. |
| **Pivot Values** | `pivotValues` | Value(s) from pivot column to be used (optional). |
| **Pivot Columns** | `pivotColumns` | Columns for pivoting. |
| **Unchanged Columns** | `unchangedColumns` | Unchanged column(s). |
| **Unpivot Columns** | `unpivotColumns` | Columns for unpivoting. |
| **Unpivot Names** | `unpivotNames` | Names for columns after unpivoting. |
