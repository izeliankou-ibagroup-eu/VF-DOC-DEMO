# Slice

## Configuration Parameters

| UI Name | Key (Code) | Description |
| :--- | :--- | :--- |
| **Name** | `name` | Provide a stage name. |
| **Mode** | `Mode` | Slice mode defines the type of behavior, whether it drops columns or keep them.<br><br> 'Drop mode' — All mentioned columns will be dropped from the result.<br><br> 'Keep mode' — All mentioned columns will be kept in the result. |
| **Columns** | `Columns` | Specify column names. |
